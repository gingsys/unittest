Valores Comunes 

En esta opción se puede crear, modificar y eliminar valores iniciales del sistema.

 

Tipos de Acceso 

Determina la información para dar acceso al Recursos, en esta opción se puede crear, modificar y eliminar los tipos de acceso del sistema.

Los tipos de acceso posibles son:

  • Simple, tiene acceso o no tiene acceso.
  • Texto, tiene o no tiene acceso, además requiere un texto informativo sobre el acceso (por ejemplo en el caso de un correo electrónico puede ingresarse el nombre de la cuenta que se desea).
  • Selección Simple, tiene o no tiene acceso, además se escoge un valor de una lista como información adicional para el Recurso.
  • Selección Múltiple, tiene o no... more 
Workflows 

En esta opción se puede crear, modificar y eliminar los flujos de trabajo a usar, los flujos son configurables.
Un Workflow, es un conjunto de acciones a realizar luego de grabar una Solicitud de Acceso.
Por cada recurso solicitado se inicia automáticamente el flujo de Aprobación y/o Notificación definido para el recurso.

 

 

Categorías 

En esta opción se podrán crear, modificar y eliminar categorías de los recursos, un ejemplo básico seria la categoría “Equipos informáticos” la cual agrupa a los recursos: acceso telefónico, equipo, equipo móvil, etc.

 

Recursos 
Permite crear, modificar y eliminar recursos, los cuales pueden tener asociados tipos de acceso específicos, como ejemplo podríamos mencionar el recurso, “Software Licenciado” y los accesos Camtasia, S10, etc.
 
Personas 
Es posible la crear, modificar y eliminar personas, a las cuales se les podrá asignar diversos tipos de permisos y/o accesos.
 
Usuarios 
Es posible la crear, modificar y eliminar personas, a las cuales se les podrá asignar diversos tipos de permisos y/o accesos.
 
Configuración de notificaciones 
 

En esta ventana podremos ingresar los parámetros iniciales como: el servidor de correo, el puerto, el usuario de envío de notificaciones y su respectiva contraseña. Recordar que sin estos parámetros no podrá funcionar el servicio de notificaciones. 

 

Jerarquía de Aprobación 

Una Jerarquía de Aprobación es una lista ordenada de Área/Cargo que permite identificar a las Personas que intervienen en workflow de aprobación.

Se ingresará las combinaciones de Área/Cargo y su respectivo orden, recordar que no podrán repetirse esta combinación por jerarquía de aprobación.
