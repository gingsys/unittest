Tipos de Acceso
Menu Configuración  Topics  Send Feedback

Determina la información para dar acceso al Recursos, en esta opción se puede crear, modificar y eliminar los tipos de acceso del sistema.

Los tipos de acceso posibles son:

  • Simple, tiene acceso o no tiene acceso.
  • Texto, tiene o no tiene acceso, además requiere un texto informativo sobre el acceso (por ejemplo en el caso de un correo electrónico puede ingresarse el nombre de la cuenta que se desea).
  • Selección Simple, tiene o no tiene acceso, además se escoge un valor de una lista como información adicional para el Recurso.
  • Selección Múltiple, tiene o no tiene acceso, además se escoge 1 o más valores de una lista como información adicional para el Recurso.

Nuevo Tipo de Acceso 

En esta opción se creará un nuevo Tipo de Acceso, más adelante se pueden ver los tipos de acceso posibles, al final de la creación presiones guardar .

 

Modificar Tipo de Acceso 

En esta opción modificará un Tipo de Acceso existente, al final de presiones para guardar los cambios, también es posible realizar una modificación dando doble click en la fila a modificar.

 

Eliminar Tipo de Acceso 

Esta opción eliminara un Tipo de Acceso, tenga en cuenta que si existe dependencias no se podrá eliminar dicho registro.
